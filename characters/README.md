# 인물 인덱스 — characters/README.md

> 소속 문서: 무소유(無所有) 설정 바이블
> 시트 양식: 외모·체형 / 말버릇·입버릇 / 대표 행동 / 자기서사 / 실제와의 간극 / 세계 적응법 / 화자와의 관계 / 대표 대사

> **중요 원칙 — 이 판단은 사내 평판이 아니다.** 이 시트에 담긴 인물 평가는 회사 내에 공유되거나 공인된 평판이 아니다. 사내에 이 인물들에 대한 집단적 평가는 존재하지 않는다. 이것은 전적으로 **화자 개인의 관찰과 발견**이다. 화자는 이 판단을 아무와도 공유하지 못한 채 혼자 짊어지고 있다. 집필 시 "다들 알지만 말하지 않는다"와 "아무도 모른다, 오직 화자만 봤다"를 혼동하지 말 것 — 이 작품은 후자에 더 가깝다. (예외: EP.1 무소유처럼 화자가 직접 공론화를 시도한 사건에 한해서만 타인의 반응(좋아요 등)이 실제로 존재한다.)
